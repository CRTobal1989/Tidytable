Este repositorio documenta el análisis exploratorio de un conjunto de datos obtenido mediante scraping que contiene información sobre sets de LEGO (lego_population_cof.csv).

El trabajo aborda la inspección inicial, limpieza y estructuración de datos, seguida de análisis descriptivos numéricos (conteos, medias, máximos y otras métricas agregadas) y visualizaciones orientadas a identificar patrones, relaciones entre variables y comportamientos relevantes. Se presta especial atención a la transparencia del razonamiento analítico, documentando las hipótesis planteadas y su validación progresiva.

El procesamiento se realiza en entorno R empleando principalmente tidytable y data.table para manipulación eficiente de datos y ggplot2 para visualización, buscando soluciones compactas, legibles y reproducibles. El resultado constituye una base analítica que refleja prácticas habituales en exploración de datos en contextos profesionales donde la calidad inicial de la información es limitada.
